Flexbox - Bac à sable

Flexbox repose son travail sur deux axes qui sont par défaut :
l’axe principal ('main axis') est disposé dans le sens horizontal [row] par défaut,
et l’axe perpendiculaire ('cross axis') dans le sens vertical [column] par défaut.
[justify-content] travaille sur l'axe principal et [align-items] sur le 'cross'.

Propriétés du conteneur flex [parent]

display: flex | inline-flex;
définit un container flex, il est block par défaut
crée un contexte flex pour tous les descendants directs
